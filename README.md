# 📖 Overview

Learning REST CRUD is a practice project created during my Full Stack Development journey to understand how RESTful applications are built using Express.js and EJS.

The application demonstrates CRUD (Create, Read, Update, Delete) operations, server-side rendering, routing, dynamic templates, and form handling while following REST principles.
